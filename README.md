## Dependencies

- bcryptjs
- body-parser
- cookie-session
- EJS
- Express
- Node.js

## Getting Started

- Install all dependencies (using the `npm install` command in the terminal).
- Reun the development web server using the `node express_server.js` command in the terminal.
